# GPA Calculator

A dynamic GPA calculator that allows students to easily calculate their Grade Point Average (GPA) based on selected subjects, grades, and credit hours.

## Features
- Add unlimited subjects dynamically
- Select grade and corresponding credit hours for each subject
- Automatically calculates GPA in real-time
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open `gpa.html` in your browser.
2. Click "Add Subject" to add a new subject row.
3. Select the grade and credit hours for each subject.
4. GPA will be calculated automatically at the bottom.
